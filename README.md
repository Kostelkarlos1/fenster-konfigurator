# Fenster-Konfigurator CH

A modern, responsive window and door configurator for the Swiss market.

## Features

- 5 Manufacturers: Schüco, Kömmerling, Aluplast, Rehau, Veka
- 6 Product Types: Fenster, Terrassentüre, Balkontüre, Haustür, Rollladen, Raffstoren
- 4 Languages: Deutsch, English, Français, Italiano
- 50+ Colors with real-time preview
- 5 Glazing Options with energy ratings
- 3D visualization with Three.js
- Real-time pricing engine with Swiss VAT
- Fully responsive mobile design

## Quick Start

npm install
npm run dev

## Tech Stack

- React 18 + TypeScript
- Three.js + React Three Fiber
- Zustand for state management
- i18next for translations
- Tailwind CSS for styling
- Vite for build tool
